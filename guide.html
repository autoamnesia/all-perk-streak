ğŸ¯ What is All Perk Streak?

All Perk Streak is a tool designed to help Dead by Daylight players track their progress in using every perk available for both Killers and Survivors. The goal is to win with every killer/survivor in a row using all perks - aiming to kill at least 3 people as killer or escape as survivor. Each perk can only be used once until you've used them all, so some characters will only use 1, 2, or 3 perks since there aren't enough perks for every character to have 4!

1. Choose Your Role

From the home page, click either Killers or Survivors to start tracking perks for that role.

2. Select a Character

Click on any character portrait to select them. Selected characters have a blue outline.

3. Assign Perks

Choose up to 4 perks from the available perk list by clicking on them. They'll appear in your perk slots at the top.

Right-click on the character when you've finished your build to mark them as completed. Completed characters have a green border.

OBS Streamer Overlay (Local Only)

What is the OBS Overlay?

The OBS overlay provides a transparent, real-time display of your current All Perk Streak progress that can be added directly to your stream. It shows your selected character and assigned perks in a clean, overlay-friendly format.

Setting Up the Overlay

âš ï¸ Local Only: The OBS overlay requires running a local server and only works when used locally on your computer. It cannot be used from the Website version.

To set up the OBS overlay:

  1. Visit the Streamer Setup Guide for detailed instructions
  2. Run the local server using start_server.bat
  3. Add a Browser Source in OBS pointing to the overlay URL
  4. Your progress will automatically sync and display in real-time

Overlay Features

  • Real-time sync: Updates automatically as you change characters and perks
  • Transparent background: Blends seamlessly with your stream
  • Clean design: Optimized for stream visibility
  • Customizable CSS: Edit the overlay styles to match your stream theme
